# cocmdPluginsToggle

A button which toggles visibility of plugins.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'plugins' | 'none';
}
```

```jsx
<Center>
  <cocmdPluginsToggle size="sm" meeting={meeting} />
  <cocmdPluginsToggle size="lg" meeting={meeting} />
  <cocmdPluginsToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
