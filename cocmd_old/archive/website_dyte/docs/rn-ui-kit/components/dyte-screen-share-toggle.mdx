# cocmdScreenShareToggle

A button which toggles your screenshare.

```jsx
<Center>
  <cocmdScreenShareToggle size="sm" meeting={meeting} />
  <cocmdScreenShareToggle size="lg" meeting={meeting} />
  <cocmdScreenShareToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
