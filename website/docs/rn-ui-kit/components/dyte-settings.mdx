# cocmdSettings

A settings component to see and change your audio/video devices as well as see
your connection quality.

```jsx
<Center>
  <h2>Desktop</h2>

  <cocmdSettings
    meeting={meeting}
    style={{ height: '480px', width: '100%', maxWidth: '720px' }}
  />

  <h2>Mobile</h2>

  <cocmdSettings
    meeting={meeting}
    size="sm"
    style={{ height: '480px', width: '100%', maxWidth: '360px' }}
  />
</Center>
```
