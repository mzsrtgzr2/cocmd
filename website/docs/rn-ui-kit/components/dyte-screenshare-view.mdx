# cocmdScreenshareView

A component which plays a participant's screenshared video.

It also allows for placement of other components similar to
`dyte-participant-tile`.

This component will not render anything if the participant hasn't start
screensharing.

```jsx
<cocmdScreenshareView participant={meeting.self} style={{ height: '480px' }}>
  <cocmdNameTag participant={meeting.self}>
    <cocmdAudioVisualizer slot="start" participant={meeting.self} />
  </cocmdNameTag>
</cocmdScreenshareView>
```
