# cocmdSettingsToggle

A button which toggles visibility of chat.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSettings: boolean;
}
```

```jsx
<Center>
  <cocmdSettingsToggle size="sm" />
  <cocmdSettingsToggle size="lg" />
  <cocmdSettingsToggle variant="horizontal" size="sm" />
</Center>
```
