# React Applications

cocmd's UI Kit provides all the necessary UI components to allow complete customization of all its UI Kit components. You can customize your meeting icons such as chat, clock, leave meeting, mic on and off, and more.

Customizing the meeting icons involve the following steps:

1. Integrate cocmd’s web SDK into your web application
2. Customize the default icon pack

## Step 1: Integrate cocmd’s web SDK into your web application

<details open>
    <summary>
        To get started with customizing the icons for your meetings, you need to first integrate cocmd's Web SDK into your web application.
    </summary>
    1. Install the package using npm or Yarn.

```bash npm2yarn
npm install @dytesdk/react-ui-kit @dytesdk/react-web-core
```

2. Initialize the cocmd client using the `usecocmdClient()` hook. It returns an array with two values, the meeting object and a function to initialize the meeting.
3. Create a meeting room using the [Create Meeting API](/api#/operations/create_meeting).
4. Generate an `authToken` using the [Add Participant API](/api#/operations/add_participant). An `authToken` is a unique token that is used to identify a user in the meeting.
5. Initialize the meeting using the `initClient()` method exposed by `usecocmdClient`.
6. Pass the meeting object to the UI Kit.

The `cocmdMeeting` component generates the default UI experience. Read more about how to customize the UI [here](/react-ui-kit/basics/components-basics).

```jsx
import { usecocmdClient } from '@dytesdk/react-web-core';
import { cocmdMeeting } from '@dytesdk/react-ui-kit';

function App() {
  const [client, initClient] = usecocmdClient();

  useEffect(() => {
    initClient({
      authToken: '<auth-token>',
      defaults: {
        audio: false,
        video: true,
      },
    });
  }, []);

  return <cocmdMeeting meeting={client} />;
}
```

</details>

## Step 2: Customize default icon pack

Now that you've integrated cocmd SDK into your application successfully, let's quickly understand how you can customize the icon set for your application.

cocmd's default icon set is available at [icons.cocmd.ai](https://icons.cocmd.ai/). You can modify and generate your custom icon set from here.

In order to replace cocmd's default icon set with your own, you can pass the link to your icon set in the `cocmdMeeting` component like so.

```jsx
 <cocmdMeeting meeting={meeting!} iconPackUrl="my-icon-pack.json" />;
```
