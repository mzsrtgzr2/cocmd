# Media

This topic compares how OpenTok and cocmd handle media publishing and subscription behavior.

## Publishing

OpenTok

```ts
publisher = OT.initPublisher('replacementElementId');
session.publish(publisher, function (error) {
  if (error) {
    console.log(error);
  } else {
    console.log('Publishing a stream.');
  }
});
```

For more information, see [Publishing a stream](https://tokbox.com/developer/guides/publish-stream/js/).

cocmd

```ts
cocmdClient.init({
  authToken: '$TOKEN',
  defaults: {
    audio: true,
    video: true,
  },
});
```
