
# cli run

Run something

## Usage

```
Usage: cli run [OPTIONS] [NAME]
```

## Options
* `name`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `name`

  


* `yes`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `-y
--yes`

  Don't ask 'are you sure' for every step


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.



## CLI Help

```
Usage: cli run [OPTIONS] [NAME]

  Run something

Options:
  -y, --yes  Don't ask 'are you sure' for every step
  --help     Show this message and exit.
```
