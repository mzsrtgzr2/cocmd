# cocmdParticipants

A component which lists all participants, with ability to run privileged actions
on each participant according to your permissions.

```jsx
<Center>
  <cocmdParticipants
    meeting={meeting}
    style={{ height: '480px', maxWidth: '320px', backgroundColor: '#000' }}
  />
</Center>
```
